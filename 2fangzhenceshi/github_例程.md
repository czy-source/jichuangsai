# 第十届集创赛“雄芯杯”FPGA异构计算方案（开源项目+评分匹配+开发策略）
## 一、赛题核心定位
典型**FPGA底层像素级流水线加速 + RISC-V硬核顶层控制**异构计算任务，无完全开箱即用开源项目，需组合高质量开源工程构建完整系统，以下为3个高度契合的开源参考工程。

## 二、高匹配度开源参考项目（附评分指标匹配分析）
### 1. 核心视觉算法参考：lauchinyuan/Verilog_IMG_PROC
纯Verilog编写的底层图像处理流水线库，适配初赛**颜色+边缘提取**底层逻辑。
- 项目地址：https://github.com/lauchinyuan/Verilog_IMG_PROC
- 核心功能：RGB2HSV、Sobel边缘检测、腐蚀/膨胀形态学滤波、直方图统计
- 赛题6项评分指标匹配度
  1. **实时性要求（≤1ms & ≤16ms）**：完全满足。纯组合逻辑+流水线寄存器设计，像素流处理延迟仅微秒级，远低于时限要求
  2. **HSV阈值可配置**：满足。自带rgb2hsv.v模块，外接比较器，通过AXI-Lite由RISC-V动态配置5种颜色阈值
  3. **形状特征区分**：部分满足。提供边缘检测+形态学处理前置能力，需补充连通域分析/霍夫变换实现形状识别
  4. **摄像头驱动**：无MIPI驱动，仅支持纯数据流处理，需外接摄像头控制IP
  5. **资源消耗**：极低。LUT/DSP占用小，远低于赛场370K LUT、1344 DSP资源上限
  6. **异构架构支持**：高解耦设计，标准vsync/hsync/de/data视频流时序，可封装AXI4-Stream IP核挂载RISC-V总线

### 2. 异构系统架构基座：Efinix-Inc/evsoc（Edge Vision SoC）
Efinix官方开源框架，适配赛场指定**Efinix WZZY（RISC-V+FPGA）**平台，解决软硬件协同底层问题。
- 项目地址：https://github.com/Efinix-Inc/evsoc
- 核心功能：RISC-V(Sapphire)硬核配置、MIPI CSI摄像头接入、DDR缓存、DMA控制、HDMI显示+OSD叠加
- 赛题6项评分指标匹配度
  1. **实时性要求**：满足。硬件加速模块挂载AXI视频流链路，帧处理天然实时
  2. **HSV配置**：无原生HSV，可将第一个项目算法植入hw_accel硬件加速模块
  3. **形状特征区分**：满足。RISC-V运行C语言，完成硬件预处理后的几何特征分析
  4. **摄像头驱动**：完美满足。自带MIPI CSI Rx IP、双摄Bayer转RGB解析，解决无ISP原始数据处理痛点
  5. **资源消耗**：基础工程仅占用30K-50K LUT，资源占比＜15%，预留充足算法开发空间
  6. **异构架构支持**：完美满足。标准RISC-V（控制/解算）+FPGA（像素流水线）架构，内置AXI4互联+C驱动层，支持显示与交互

### 3. 目标跟踪与特征识别参考：kaangursoy1/Real-Time-Image-Processing-on-Zynq-FPGA
基于Xilinx Zynq的工业分拣视觉项目，硬件处理逻辑与赛题100%契合。
- 项目地址：https://github.com/kaangursoy1/Real-Time-Image-Processing-on-Zynq-FPGA-PES-DHL-Project-
- 赛题6项评分指标匹配度
  1. **实时性要求**：满足。标准视频流流水线设计，行缓存级延迟
  2. **颜色+质心识别**：完美满足。实现RGB→HSV→色彩阈值过滤→质心坐标计算全流程
  3. **形状+尺寸识别**：满足。支持边界框提取+面积统计，通过**长宽比+填充率**区分正方体/圆柱/锥体，配合标定实现尺寸测量
  4. **摄像头驱动**：无原生MIPI驱动，可复用evsoc摄像头方案
  5. **资源消耗**：DSP占用可控，赛场资源完全满足
  6. **异构架构支持**：满足。提供处理器读取AXI总线寄存器逻辑，C语言控制逻辑可直接移植RISC-V

## 三、赛题致胜开发策略
### 1. 框架搭建（优先复用evsoc）
不手写AXI总线与MIPI解析，直接基于Efinix evsoc工程，打通**MIPI双摄输入→DDR帧缓存→HDMI/OSD显示**全通路。

### 2. 颜色+质心识别（初赛20分硬指标）
在视频流写入DDR前，串联Verilog_IMG_PROC的rgb2hsv模块；
通过AXI-Lite挂载5组阈值寄存器，由RISC-V C语言动态配置；
硬件累加命中像素X/Y平均值，直接输出目标质心坐标。

### 3. 形状识别（初赛18分关键，最优软硬协同方案）
- **避坑**：不推荐FPGA硬件实现复杂霍夫变换（耗资源、易时序违例）
- **推荐方案**：
  FPGA端：完成Sobel边缘检测+最小外接矩形+面积统计；
  RISC-V端：通过C语言计算**填充率（面积/外接矩形面积）**，快速区分形状：
  正方体填充率≈1、圆柱≈π/4、锥体≈0.5；
  优势：满足≤16ms实时性，大幅降低开发难度，符合评委认可的软硬协同设计思路。

## 四、方案总结
1. 以**Efinix evsoc**为系统框架，解决摄像头、显示、总线、RISC-V控制核心问题；
2. 以**Verilog_IMG_PROC**为底层算法核心，实现颜色提取与边缘检测；
3. 以**工业分拣视觉项目**为参考，完成质心、形状、尺寸识别的软硬协同设计；
4. 全程满足实时性、资源、异构架构要求，适配集创赛“雄芯杯”全评分维度。
